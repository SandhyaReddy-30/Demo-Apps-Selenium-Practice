FAQ Accordion
What is Selenium? ▼
Selenium is an open-source framework for automating web browsers. It supports multiple programming languages including Java, Python, C#, and more. It's widely used for web application testing.
What is WebDriver? ▼
WebDriver is a remote control interface that enables introspection and control of user agents. It provides a platform and language-neutral wire protocol as a way for out-of-process programs to remotely instruct the behavior of web browsers.
What are locators in Selenium? ▼
Locators are used to identify HTML elements on a webpage. Selenium supports: id, name, className, tagName, linkText, partialLinkText, cssSelector, and xpath.
What is implicit wait vs explicit wait? ▼
Implicit wait tells WebDriver to poll the DOM for a certain duration when trying to find any element. Explicit wait makes WebDriver wait for a specific condition before proceeding further in the code.
